# Fundamentos del Desarrollo Web DEVF
Este repositorio contiene las prácticas del módulo sobre **Fundamentos del Desarrollo Web** como parte del **Course Frontend asistido con IA de DEV.F**.

## 🎯 Objetivo del Proyecto
Crear páginas web sencillas implementando buenas prácticas de separación de código (HTML para estructura y CSS para diseño visual) y publicar el resultado final en la web utilizando GitHub Pages.

## 🛠️ Tecnologías y Herramientas Utilizadas
- **HTML5:** Estructura semántica del documento.
- **CSS3:** Estilos visuales a través de un archivo externo (`style.css`).
- **Git & GitHub:** Control de versiones y alojamiento remoto del código.
- **GitHub Pages:** Plataforma de despliegue para la página web estática.
